# @form-validator-js/validators

Built-in validators for [`form-validator-js`](https://github.com/mellonis/form-validator-js): `required`, `minLength`, `maxLength`, `pattern`, `equalsTo`, `checkedCount`, `numeric`, `min`, `max`, `step`.

Peer-depends on [`@form-validator-js/core`](https://www.npmjs.com/package/@form-validator-js/core), pinned to the exact same version.

## Install

```sh
npm install @form-validator-js/core @form-validator-js/validators
```

## Documentation

See the [project README](https://github.com/mellonis/form-validator-js#readme) for the DSL syntax, validator details, and examples.
